# jeet-image
一个基于OpenResty + GraphicsMagick实现的图片服务

## 环境
 - 系统环境：MacOS 10.12
 - OpenResty版本：1.11.2.2
 - GraphicsMagick版本：1.3.25

## 功能
 - 图片上传
 - 动态生成缩略图
 - FastDFS存储
